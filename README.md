## Titanic EDA Project

Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python, Pandas, Seaborn, and Matplotlib.

Questions Analyzed

1. Did women have a higher survival rate than men?
2. Did passenger class affect survival?
3. Did age affect survival and differ by sex?

Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib

Results

- Women had a higher survival rate than men.
- First-class passengers survived more often.
- Age and gender influenced survival patterns.
